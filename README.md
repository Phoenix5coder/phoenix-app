Phoenix App
A full-stack Twitter/X-style social media application built with React and Supabase, featuring a complete suite of social interactions including real-time feeds, user authentication, and media sharing.
Features

User Authentication — Secure sign up, login, and session management via Supabase Auth
Posts & Feed — Create, view, and delete posts in a dynamic, real-time feed
Likes & Comments — Engage with posts through likes and threaded comments
Follow System — Follow and unfollow users to customize your feed
Media Uploads — Share images directly in posts via Supabase Storage
